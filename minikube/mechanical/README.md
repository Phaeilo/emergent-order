These are the mechanical parts for the "minikube" LED cube. All parts are fully 3D printable.

## Hardware Required

- **M3 screws:** 24× screws, approximately 8mm length
  - Used to securely connect sticks with corners

## Printing Guidelines

- **Material:** PLA or PETG recommended
- **Layer height:** 0.2mm
- **Settings:** Standard settings should work fine

## Assembly Instructions

1. **Print all parts:**
   - 12× Stick
   - 7× Corner
   - 1× Base

2. **Assemble the cube frame:**
   - Connect sticks to corners using M3 screws to form the cube structure
   - Secure the base to the bottom of the cube frame

4. **Thread the LED string:**
   - Start at the base: secure the start of the string with a loop, or zip-tie
   - Thread the string using a chaotic pattern to fill the volume
   - Secure the end of the strip
